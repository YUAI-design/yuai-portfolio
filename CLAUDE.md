# YUAI Portfolio Project Memory

## プロジェクト概要
- **会社名**: ユーアイ (YUAI)
- **ドメイン**: yuai.design
- **目的**: ホームページ作成・ウェブアプリ開発の副業から起業へ
- **現在フェーズ**: ポートフォリオサイト構築

## 技術スタック
- **フロントエンド**: HTML5, CSS3, JavaScript (ES6+)
- **バージョン管理**: Git, GitHub
- **デプロイ**: Vercel
- **開発環境**: VS Code + WSL + Claude Code

## 開発方針
- **デザイン**: モダン、シンプル、プロフェッショナル
- **レスポンシブ**: モバイルファースト
- **アクセシビリティ**: WCAG 2.1 AA準拠
- **SEO**: 基本的な最適化実装
- **パフォーマンス**: 軽量、高速読み込み

## サイト構成
1. **ヘッダー**: ナビゲーション、ロゴ
2. **ヒーローセクション**: 会社紹介、キャッチコピー
3. **サービス紹介**: ホームページ制作、ウェブアプリ開発
4. **実績・ポートフォリオ**: 今後の案件用
5. **お問い合わせ**: フォーム、連絡先
6. **フッター**: 会社情報、SNSリンク

## ブランドガイドライン
- **カラーパレット**: 
  - プライマリ: ダークブルー系
  - セカンダリ: ホワイト、グレー
  - アクセント: 明るいブルーまたはグリーン
- **フォント**: モダンなサンセリフ (Google Fonts推奨)
- **トーン**: プロフェッショナル、信頼感、革新的

## コーディング規約
- **HTML**: セマンティックタグ使用
- **CSS**: 
  - BEM命名規則
  - CSS Custom Properties使用
  - Flexbox/Grid レイアウト
- **JavaScript**: 
  - ES6+ 構文
  - 関数型プログラミング志向
  - コメント必須
